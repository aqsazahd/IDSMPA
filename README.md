# IDSPMA: Interpretable Drug Sensitivity Prediction with Meta-Heuristic Algorithm

This repository provides the full reproducibility package for the paper:

> **Aqsa Zahid**, *IDSPMA: Interpretable Drug Sensitivity Prediction with Meta-Heuristic Algorithm*, IEEE Access, 2025.  
> [Manuscript under review]

The IDSPMA framework integrates **Information Gain (IG)**, **Particle Swarm Optimization (PSO)**, and a **Deep Neural Network (DNN)** for predicting drug sensitivity (IC50). The model is further interpreted using **SHAP** analysis to identify biologically relevant genes.
